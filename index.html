Welcome to GitHub Pages.

Your task is to implement a simple contact list. This involves creating two Javascript objects:

Contact(). The Contact constructor should accept the following String arguments: firstName, lastName, address, zipcode, and telephone number. All of these arguments should be accessible (i.e. contact.firstName should return the firstName string). In addition, the function should define the following functions:

call() indicates you are calling that contact. It updates an internal counter and returns true indicating that the call was successful.

numCalls() returns an integer indicating the number of times that contact has been called.

ContactList(). The ContactList() constructor accepts no arguments, but defines a number of functions.

addContact() accepts a Contact instance and adds it to an internal data structure.

numContacts() returns the number of contacts in this ContactList().

findContact() is passed a String and returns all of the Contacts with that lastName.

deleteContact() is passed a firstName and lastName and deletes all matching Contacts.

listContacts() returns a string containing the contacts sorted by lastName.
